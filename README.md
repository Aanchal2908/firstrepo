# Simple Navbar (HTML + CSS)

A basic responsive navigation bar built using only HTML and CSS.

## Files

- `index.html` - Contains both the navbar HTML structure and CSS styles.

## How to Run

1. Open `index.html` in any web browser.

## Navbar Features

- Brand/logo text on the left
- Navigation links on the right
- Hover effect on links
- Wraps nicely on smaller screen widths

## Customize

- Change brand text in the `.brand` anchor
- Add/remove menu items in the `.nav-links` list
- Update colors in `.navbar` and `.nav-links a:hover`
